
Stack AI Foundation – AI Assisted Coding – Level  2AI-Assisted Coding Assignment: Customer 360 Analytics AppAssignment IntentThis assignment evaluates whether learners can use an AI coding assistant effectively across a realistic software delivery workflow: code understanding, code generation, data transformation, testing, documentation, app generation, review, and packaging.1. Problem StatementA business stakeholder wants a Customer 360 analytics mini-application that combines customer master data, order/revenue data, and customer support ticket data. The existing pipeline code is incomplete and unreliable. Your task is to use GitHub Copilot or another approved AI coding assistant to complete the pipeline, create tests, generate documentation, and build a small Streamlit app for visualization.The final solution should be an executable project folder that another team member can run using the README instructions.2.
